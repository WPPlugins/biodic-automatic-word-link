== Description ==

BioDic Automatic word link es un plugin para WordPres que habilita la posibilidad de parsear tus entradas y añadir de forma automática enlaces del Diccionario de Biología BioDic, así tus lectores podrán aprender a la vez que leen tus entradas.
El plugin cuenta con la opción de mostrar un tooltip para que el usuario pueda leer rápidamente la definición completa o parte de ella, directamente en tu web.


== Installation ==

1. Sube el contenido del zip a la carpeta `/wp-content/plugins/` 
2. Activa el plugin a través del menú 'Plugins' del panel de WordPress
3. Configura las opciones de visualización en la seccion 'BioDic AWL' del panel de control de WordPres
